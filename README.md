# Claude Code Plugins

Personal collection of Claude Code plugins.

## Installation

Add this marketplace:

```bash
claude plugin marketplace add github:maxnoller/claude-code-plugins
```

Then install plugins:

```bash
claude plugin add using-jules
```

## Plugins

### using-jules

Guide for using Google Jules, the asynchronous AI coding agent. Helps with:

- Writing effective prompts for Jules
- Using the `jules` CLI commands
- Deciding when to delegate to Jules vs keep in Claude Code
- Workflow patterns for batch processing and GitHub integration
